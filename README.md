# Autism Traffic Light Timer

A mobile application designed to support autistic children using simple and predictable visual timers based on traffic light signals.

The app helps caregivers create structured routines, activity transitions, focus sessions, break periods, and calming moments through customizable green and red countdown timers.

---

## Features

### Configurable Traffic Light Timers
- Set custom durations for:
  - Green light
  - Red light
- Visual countdown display
- Easy-to-understand interface for children

### PIN Protection
- Lock timer controls and settings
- Prevent accidental or unauthorized changes
- Configurable security PIN

### Accessibility Focused
- Simple and minimal interface
- Large visual indicators
- Designed for routine predictability and reduced overstimulation

### Use Cases
- Study sessions
- Bedtime routines
- Screen time control
- Therapy support
- Classroom activities
- Calm-down periods
- Daily transitions

---

## Goal

This project was created to provide a simple, accessible, and reliable visual timer tool for autistic children and their caregivers, helping improve routine understanding, independence, and time management.
